Создайте функцию-конструктор Box (коробка) для создания объекта со свойствами Height (высота), 
        Width (ширина), Depth (глубина) и Material (материал). 
        Также, в объекте должны быть методы Volume – для получения объема и equals() – 
        для сравнения объектов, объекты должны считаться равными, если содержат одинаковые значения в свойстве Material.
